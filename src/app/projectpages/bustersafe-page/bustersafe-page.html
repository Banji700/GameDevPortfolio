Buster Safe

For this game I was tasked to recreate the game Buster Safe as part of a job interview. A video of what the game should look like and the assets were provided for me to use all I had to do was coding and put it together.


Process

First thing I did was lay out the main assets and then I put them together so I could have visual idea for the game.

The main part I focused on was the wheel part becasue I found that part quite challenging. But I was able to make it work so that the wheel will spin then land on a number.

Wheel Spin Script

Wheel Spinning Video

You can only get four spins and if you land on one number you cant land on the same number again. This makes sure that you can get a multiplier on the fourth spin if you didnt get one on the first three spins. When the wheel lands on a number a vault will open and then a multiplier will be revealed according to the number the wheel landed on and if you get two of the same multiplier you essentialy win the game.

Vault Open Video

I also added some text in the top box area to make the player aware of what is going on so if you are still spinning the wheel the words "spinning" will be displayed but also if you land on a vault the number of that vault will be displayed

But when I was able to get it done everything else was easy for me and it was fun to make because this was something new to me.
