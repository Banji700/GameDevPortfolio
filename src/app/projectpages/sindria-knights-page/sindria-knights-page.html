Sindria Knights

For this game I had mainly wanted to try some new mechanics and learn a new way of doing things in Unity because I feel like it would be better for me to learn different ways of doing things. This game is still in development because there’s a lot of things I still want to add.




Inspiration

I took inspiration from these types of games because when I was younger these are the type of games I would play when I didn’t have a console.




Process

I first started by getting character models and animations from the unity asset store. I then proceeded to start giving my character movement using the way I’m familiar with which is writing a mono behaviour script and adding it to the character.

The other way I wanted to try was using state machines. At first, I didn’t understand this method and found it difficult to use but once I understood how to use this method it became easy. State machines use different scripts to make the character do different things and although it is a longer way of doing thing, I still think it is very effective and so I decided to use this method for the rest of the game.


Using State Machines

Using state machines allows you to switch states of the character so if the player was moving that would be the moving state and if the player was attacking that would be the attacking state , you can have more states depending on what you want your character to do.
