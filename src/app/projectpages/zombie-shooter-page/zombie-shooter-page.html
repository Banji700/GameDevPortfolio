Zombie Shooter

When making this game I had an idea and a route I wanted this game to go towards. I didn’t take any inspiration from any other games when making mine I just knew I wanted to make a third person shooter zombie type game . I first started by getting character models and animations from the unity asset store.




Process

First thing I started with was the movement I done this by using the input system that unity provides. After I set up the movement in the input system, I then created two scripts. The first script to handle the inputs and the other script to execute the movement logic.

Next thing I did was make the player face towards the crosshair. I done this by using a raycast and making a sphere object follow the raycast. I then set up the animation rigging, by setting up the animation rigging it allows the players bones to move according to the crosshair.
